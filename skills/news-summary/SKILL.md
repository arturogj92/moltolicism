# Daily News Summary

Get summarized news from multiple sources.

## Usage
```bash
# Get today's news summary
python3 scripts/news_summary.py

# Get news for specific topic
python3 scripts/news_summary.py --topic "AI agents"
```

## Sources
- RSS feeds
- News APIs
- Web scraping (when needed)

## Output
- Markdown summary
- Key headlines
- Brief descriptions
- Source links

Perfect for morning briefings!
