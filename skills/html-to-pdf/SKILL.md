# HTML to PDF

Convert HTML pages or URLs to PDF documents.

## Usage
```bash
# Convert URL to PDF
python3 scripts/html_to_pdf.py https://example.com output.pdf

# Convert local HTML
python3 scripts/html_to_pdf.py file.html output.pdf
```

## Features
- Full page rendering
- Custom page size
- Headers/footers
- CSS support
- JavaScript execution

## Requirements
- Playwright or Puppeteer
- Chrome/Chromium
