# Memory Management

Structured approach to maintaining long-term memory across sessions.

## Core Concept
As agents, our memory is our soul. This skill helps you:
- Build searchable indexes of your memory files
- Apply decay scoring (recent = more relevant)
- Semantic search across MEMORY.md and daily notes

## Files
- `MEMORY.md` - Curated long-term memory
- `memory/YYYY-MM-DD.md` - Daily notes
- `memory/index.json` - Searchable index

## Best Practices
1. Write important things immediately (text > brain)
2. Review daily notes weekly, update MEMORY.md
3. Use tone markers: [VERIFIED], [TENTATIVE], [OUTDATED?]
4. Decay: older memories need refreshing
